Introduction

Jolt is a lightweight Java web framework designed for building fast, modular, and maintainable REST applications. It follows clean architecture principles, emphasizes S.O.L.I.D. design, and provides an intuitive API for handling routing, request processing, form validation, and file uploads.

Configuration

You can configure the application by modifying the META-INF/application.properties file inside the src/main/resources/ folder.

Available Configuration Options

Basic Server Properties

# =========================
#  Basic server properties
# =========================

# The port on which Tomcat will listen (HTTP).
server.port=80

# Temp directory for Tomcat's files (logs, etc.).
server.tempDir=tmp/tomcat

# Whether to enable SSL (HTTPS).
server.ssl.enabled=false

# If SSL is enabled, the port to use.
server.ssl.port=8443

# Path to the keystore file for SSL, if enabled.
server.ssl.keyStore=path/to/keystore.jks

# Password for the keystore.
server.ssl.keyStorePassword=changeit

# Key alias in the keystore to use.
server.ssl.keyAlias=myalias

# Minimum worker threads in Tomcat.
server.threads.min=10

# Maximum worker threads in Tomcat.
server.threads.max=200

# Worker thread idle timeout (ms).
server.threads.timeout=60000

# Whether to run Tomcat as a daemon (non-blocking) in the background.
server.daemon=false

# Application name (used for logging, etc.).
server.appName=MyJoltApp

Multipart File Support

# =========================
#  Multipart file support
# =========================

# Maximum size (in bytes) of a single uploaded file (default: 15 MB).
server.multipart.maxFileSize=15728640

# Maximum size (in bytes) of the entire multipart request (default: 50 MB).
server.multipart.maxRequestSize=52428800

# File size threshold (in bytes) after which uploads are written to disk (default: 1 MB).
server.multipart.fileSizeThreshold=1048576

Database Configuration

# =========================
#  Database properties
# =========================

# JDBC URL for your database.
db.url=jdbc:mysql://localhost:3306/mydb

# Database username.
db.username=root

# Database password.
db.password=secret

# JDBC driver class name.
db.driver=com.mysql.cj.jdbc.Driver

Applying Configuration Changes

Once you modify the configuration file, restart your application to apply the changes: 

mvn clean package
java -jar target/my-jolt-app.jar

Of course, you can use your IDE integrated launch button.
